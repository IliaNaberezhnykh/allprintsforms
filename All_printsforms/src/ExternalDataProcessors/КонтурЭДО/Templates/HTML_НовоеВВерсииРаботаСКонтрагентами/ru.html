Обновилась форма работы с контрагентами. Ниже описаны основные изменения:

1. Разделы "Ваши контрагенты", "Приглашают вас" и "Ожидается ответ" объединены в один общий список “Ваши контрагенты”. Контрагенты, по которым еще не завершена работа с приглашениями, расположены в верхней части списка. Дополнительно они будут отмечены иконками, которые позволяют прочитать текст приглашения.

2. Отправка приглашений новым контрагентам вынесена в отдельный раздел, к которому можно перейти по кнопке “Поиск новых контрагентов”. В нем выводятся только те контрагенты, которые были найдены в Диадоке по данным контрагентов из 1С. Ликвидированные и заблокированные контрагенты отмечены соответствующим статусом и расположены в нижней части списка.

3. Добавлена строка быстрого поиска, которая позволяет находить контрагента по известной вам информации о нем. Поиск работает по следующим полям: Контрагент в Диадоке, ИНН, КПП, Идентификатор участника ЭДО.

4. Добавлены возможности:

  • Сохранение списка контрагентов при помощи команды “Вывести список”
  • Изменение состава колонок в списке контрагентов
  • Массовое удаление сопоставлений между контрагентами Диадока и 1С
  • Редактирование текста приглашения контрагентов к обмену документами
  • Просмотр даты изменения статуса отношений с контрагентом.

5. Групповое сопоставление и обновление данных с сервера Диадока разделено на две части: для каждого списка отдельно.

Подробнее про изменения смотрите в инструкции по настройке контрагентов.
